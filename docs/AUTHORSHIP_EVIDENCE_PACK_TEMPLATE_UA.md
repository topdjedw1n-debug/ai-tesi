# Evidence pack для однієї академічної роботи

Цей шаблон потрібен, щоб після кожної роботи було видно не лише фінальний
відсоток у Compilatio, а й шлях: що було згенеровано, що додано автором,
що перевірив редактор і чому роботу можна або не можна віддавати.

Не додавайте сюди приватні дані клієнта, імена, логіни, секрети або повний
