поміксувати 4-5 моделей Anthropic × моделі GPT, прогнати, зробити папку

## Чому цей експеримент

Док 10 (Opus-писар) чесно зафейлив AI-детект: 42–70% проти порога 35,
причому гуманізація тією ж моделлю ПОГІРШУВАЛА скор (53.6→70.3).
Дві механічні знахідки, які робили попередні спроби гуманізації фіктивними:

1. **Гуманізатор завжди успадковує модель писаря** — та сама модель
   переписує свій текст у свій же стиль → детектор бачить ще типовіший
   почерк (generator.py / background_jobs.py передають provider/model
   писаря в Humanizer).
2. **«Прогресивна температура» multi-pass мертва на Claude 4+/5** —
   ці моделі не приймають temperature (humanizer шле її лише для
   claude-3*), тож 2-3 «різні» паси на Opus/Sonnet — це буквально
   той самий виклик повторно.

Гіпотеза: крос-модельна гуманізація (інша родина моделей) ламає
статистичний відбиток писаря; вибір комбо калібрувати ПО ДЕТЕКТОРУ
із запасом (ціль ≤25, не «впритул до 35»).

## Фаза 1 — скринінг гуманізаторів (дешева, без повних прогонів)

Мета: звузити матрицю 6×8 до 3-4 комбо для повних прогонів.

Дизайн:
- **Тексти-писарі (6):** по одній секції ~450 слів італійською на тему
  doc 9 («Recessione economica e risposte politiche», COVID-економіка
  США) з реальними цитатами з пака doc 9 ([Wang2023], [Yin2023],
  [Balcılar2020], [Markham2024]…): gpt-5.5, gpt-4, gpt-4-turbo,
  claude-opus-4-8, claude-sonnet-5 + **реальна секція 2 doc 9**
  (gpt-4-пайплайн, GPTZero 34.6 — найгірша з тих, що пройшли).
- **Гуманізатори (7 + baseline):** none (baseline), gpt-5.5, gpt-4,
  gpt-4-turbo, gpt-3.5-turbo, claude-sonnet-5, claude-haiku-4-5,
  claude-opus-4-8 (відомий-поганий контроль).
- Один pass продакшн-промптом (PromptBuilder.build_humanization_prompt,
  language=it), збереження цитат перевіряється як у проді.
- Замір: GPTZero на кожен вихід (6 baseline + 42 humanized = 48 сканів).
- Фіксуємо: AI-скор, збереження цитат, мовний дрейф, токени/вартість.

Критерії відбору комбо у Фазу 2:
1. AI-скор ≤25 (запас до порога 35);
2. цитати збережені (Humanizer не відкотив на оригінал);
3. без мовного дрейфу;
4. серед фіналістів обовʼязково: 1 повністю-OpenAI, 1 повністю-Anthropic
   (якщо проходять), 1 крос-родинний; + контроль «gpt-4 без гуманізації».

Оцінка вартості Фази 1: ~$2–3 LLM + ~48 GPTZero-сканів.
