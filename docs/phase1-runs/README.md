# Phase 1 Proof Runs

This directory stores one sanitized quality run report per real internal proof run.

Do not include client names, private files, detector account details, raw client
instructions, secrets, or private environment values.

Phase 1 is not complete until 2-3 real reports are filled and the go/no-go
decision is updated in `docs/PHASE1_GO_NO_GO_DECISION.md`.
