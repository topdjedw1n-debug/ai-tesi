# CLAUDE.md

## Design System
Always read `DESIGN.md` before making any visual or UI decisions.
All font choices, colors, spacing, and aesthetic direction are defined there.
Direction is **Scholarly Press** — warm paper + academic green `#0f6e56`, Literata + Source Sans 3 + JetBrains Mono (all Cyrillic). Never use corporate blue as a brand accent — that is the convergence the design deliberately avoids.
Do not deviate without explicit user approval. In QA mode, flag any code that doesn't match `DESIGN.md` (especially the deprecated default Tailwind blue `primary`).
